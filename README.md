# Tic Tac Toe

## Tech Stack
- React
- Typescript
- Redux Toolkit
- React Query
- Styled Components


## Usage

```bash
# Required if you want a repository and work with Git hooks

git init

yarn install

yarn dev
```

## Available commands

Run in development mode

```bash
  yarn dev
```

Create production build

```bash
  yarn build
```

Run ESLint linting

```bash
  yarn lint
```

Run Prettier formatting

```bash
  yarn format
```

Run TypeScript compiling

```bash
  yarn compile
```

Serve production build locally

```bash
  yarn preview
```
